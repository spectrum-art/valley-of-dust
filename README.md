# Valley of Dust

A top-down 2D browser game built with Three.js, ES6 modules & Vite.

## Setup

npm install
npm run dev

Open your browser to the URL Vite prints (usually http://localhost:5173).

## Controls:

- Arrow keys to move.
- Collect dust puffs to advance levels.
- Avoid cleaners.
- Spacebar on start/win screens.
